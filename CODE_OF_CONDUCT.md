# Code of Conduct

This project adopts the Contributor Covenant, version 2.1, as its code of
conduct. The full text is available at
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In summary: be respectful, assume good faith, focus criticism on code and
ideas rather than people, and accept that maintainers may moderate
contributions and discussions to keep the project welcoming.

Instances of unacceptable behaviour may be reported to the maintainer at
the contact address listed on the repository profile. Reports are handled
confidentially.
